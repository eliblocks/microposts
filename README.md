# README

Demo Microposts App

### Requirements
  - Postgres
  - Ruby 3.2.2
  - Bun

To Install bun with Homebrew:

1. ```brew tap oven-sh/bun```
2. ```brew install bun```

### Running the project
Clone the repo and cd into it.

1. ```bundle install```
2. ```bun install```
2. ```bundle exec rails db:create```
3. ```bundle exec rails db:migrate```
4. ```bin/dev```

Then open your browser to localhost:3000
